## Clase HashMap

```java
java.lang.Object
             ↓
java.util.AbstractMap<K,V>
             ↓
java.util.HashMap<K,V>
```

### Parámetros de tipo:

```java
K - El tipo de claves <key> mantenidas por la Interface mapa <Map>
V - l tipo de valores <value> asignados por la Interface mapa <Map>
```
